SafeDNS Templates

API Documentation

Heirarchy

graph TD SafeDns --> Template --> Record style Template fill:#ffbbad


Listing Templates

# Subject to pagination.
templates = safedns.templates.list()

# Bypass pagination.
templates = safedns.templates.list(all=True)

Getting a Template

template = safedns.templates.get('my-template')

Creating a Template

template = safedns.templates.create(name='my-template', default=True)

Updating a Template

template.description = 'My new description.'
template.default = False
template.save()

Deleting a Template

template.delete()
