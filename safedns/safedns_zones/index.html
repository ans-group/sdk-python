SafeDNS Zones

API Documentation

Heirarchy

graph TD SafeDns --> Zone --> Record Zone --> Note style Zone fill:#ffbbad


Listing Zones

# Subject to pagination.
zones = safedns.zones.list()

# Bypass pagination.
zones = safedns.zones.list(all=True)

Getting a Zone

zone = safedns.zones.get('my-zone')

Creating a Zone

zone = safedns.zones.create(name='my-zone', description='My first zone.')

Updating a Zone

zone.description = 'My new description.'
zone.save()

Deleting a Zone

zone.delete()
